SageMath, Inc. Pricing (April 2, 2015)

« Other Policies

SageMathCloud charges for usage on a monthly basis, using the following pricing. Listed prices are in US dollars.

When charging in local currency, SageMath, Inc. will convert the prices listed into local currency using the conversion rates published by leading financial institutions.

Project types

Warning: The following prices are very preliminary and no product is currently available for purchase, as of April 2, 2015.

Pricing for each project type

Project type Commercial use1 Network access2 Idle timeout3 Disk space4 Speed5 Memory6 Price per month7
free-1 No No 1 hour 2GB 1 shared Opteron CPU 2GB RAM shared $0/month
shared-1 Yes Yes 24 hours 5GB 1 shared Intel CPU 2GB RAM shared $7.50/month
shared-2 Yes Yes 1 week 10GB 2 shared Intel CPU 3.75GB RAM shared $15/month
shared-4 Yes Yes infinite 50GB 4 shared Intel CPU 7.5GB RAM shared $30/month
dedicated-1 Yes Yes infinite 50GB 1 dedicated Intel CPU 3.75GB dedicated $60/month
  1. Commercial use: free-1 projects may only be used for academic and non-commercial use.
  2. Network access: Access from your project to the outside Internet, so you can pull files from external data sources easily.
  3. Idle timeout: Your project will stop running if no files are edited for this amount of time.
  4. Disk space: The total mount of disk space that your project may use. This does not count snapshots and other backups.
  5. Speed: The Opteron processors are significantly slower than the Xeon's (often a factor of 2). Shared processors may be shared with other users, so you are not guaranteed to always get the full speed.
  6. Memory: The total amount of memory (RAM) that your project may use. Shared memory is shared with other users, so you are not guaranteed to always get the full amount of memory.
  7. Price per month: Prices are not prorated. If you sign up you will be charged for a complete month.
Further Notes:
  • All free-1 projects run on a single crowded pool of computers, whereas the share projects run on their own completely separate pool of much less crowded computers.
